So erstellst du einen Job-Listing:

-- Dateinamen:
Der Name der News-Datei ist wichtig! Zuerst steht das Datum, welches auf der Website als Veröffentlichkeitsdatum
angegeben sein soll, danach ein aussagekräfties Schlagwort, z.b. der Arbeitgeber.
Beide Teile sind durch einen Unterstrich getrennt.
Beispiel: 2024-12-31_ifgi.html
Schema: [yyyy]-[mm]-[dd]_[schlagwort].html

-- Inhalt
Der Inhalt darf mit HTML formatiert werden. Empfohlen werden h2 und h3 zu nutzen.
Bitte NICHT h1 nutzen, da das mit dem Template auf dem Frontend in Konflikt kommt.
Einen Absatz pro div-Container.

-- Sicherheit
Im Frontend wird der ausgelieferte HTML-Text sanitized, d.h. <script> oder so was ist nicht möglich.
