So erstellst du einen News-Artikel:

-- Dateinamen:
Der Name der News-Datei ist wichtig! Zuerst steht das Datum, welches auf der Website als Veröffentlichkeitsdatum
angegeben sein soll, danach ein aussagekräfties Schlagwort. Beide Teile sind durch einen Unterstrich getrennt.
Beispiel: 2024-12-31_template.html
Schema: [yyyy]-[mm]-[dd]_[schlagwort].html

-- Inhalt
Der Inhalt darf mit HTML formatiert werden. Empfohlen werden h1, h2 und h3 zu nutzen.
Einen Absatz pro div-Container.

-- Sicherheit
Im Frontend wird der ausgelieferte HTML-Text sanitized, d.h. <script> oder so was ist nicht möglich.
